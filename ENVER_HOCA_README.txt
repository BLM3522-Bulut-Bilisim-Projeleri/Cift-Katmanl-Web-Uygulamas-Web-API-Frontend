Bu projede aşağıdaki adımları gerçekleştirdim:

- Backend tarafını Java (Spring Boot) ile ben yazdım. Ticket tablosunu oluşturarak CRUD işlemlerini gerçekleştiren RESTful API geliştirdim.
- Frontend tarafını React.js (Vite ile) kullanarak ben kodladım. Kullanıcı arayüzünde taleplerin listelenmesi, filtrelenmesi, düzenlenmesi gibi işlemleri sağladım.
- Veritabanını PostgreSQL ile kurdum ve Spring üzerinden bağlantısını yapılandırdım.
- Tüm sistemi AWS EC2 (Amazon Linux) üzerinde kurdum. RDS veya farklı bir dış servis kullanmadım.
- Backend ve frontend uygulamalarını EC2 içinde çalıştırarak internetten erişilebilir hale getirdim.
- Güvenlik kurallarını oluşturarak gerekli port (8080, 5174) ayarlamalarını yaptım.
- Bu projeyi **ücretsiz AWS kullanım sınırları içinde** gerçekleştirdim. Bu nedenle **sunucularım şu anda aktif değil**, sistem durdurulmuş durumda.
